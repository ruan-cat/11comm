# 通用表单组件

本项目的通用表单，旨在于为各个业务组件提供表单，以及为表格头的搜索栏提供支持。

设计范畴仅限于表单的基础行为，比如配置数据渲染表单，表单校验，表单项布局等。

本组件不会提供，或封装更多的功能。

一般仅考虑在实现业务表单时使用本组件。
